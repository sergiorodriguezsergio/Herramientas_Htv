# Herramientas Htv

Este proyecto centraliza las automatizaciones y herramientas personalizadas para el flujo de trabajo de Huelva TV.

## Automatizaciones Incluidas

### 1. Actualización Pick and Roll
Automatiza la actualización del archivo Excel de rotulación deportiva (`DATOS PICK AND ROLL.xlsx`).
- **Script**: `scripts/actualizar_pick_and_roll.py`
- **Categorías**: N1 Andaluza (Grupo A) y Tercera FEB (Grupo D-B).

### 2. Verificador de Textos
Skill para contrastar textos con información real en la web, prestando atención a nombres propios.

## Cómo usar las Skills
Simplemente menciona la tarea (ej: "Actualiza el Pick and Roll" o "Verifica este texto") y Antigravity usará las configuraciones de este proyecto.
